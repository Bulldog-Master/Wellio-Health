## D. Animation Notes (For PPTX Version)

### Recommended Transitions
- Fade-ups (preferred)
- Soft parallax on mockups
- Subtle scale on entry

### Avoid
- Spinning animations
- Harsh motion
- Bounce effects

### Timing
- 0.3s fade duration
- 0.5s delay between elements
- Auto-advance: OFF (manual control)
