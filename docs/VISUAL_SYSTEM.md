# Wellio — Visual System Recommendations

Brand visual guidelines for consistent design across all touchpoints.

---

## Color System

### Primary Colors

| Name | Hex | Usage |
|------|-----|-------|
| Deep Navy | #0D0F14 | Primary background, dark mode |
| Violet | #7C3AED | Primary accent, CTAs, highlights |
| Cyan | #22D3EE | Secondary accent, success states |

| Name | Hex | Usage |
|------|-----|-------|
| White | #FFFFFF | Text on dark, light backgrounds |
| Gray 100 | #F3F4F6 | Light backgrounds |
| Gray 400 | #9CA3AF | Secondary text |
| Gray 600 | #4B5563 | Muted text |
| Gray 900 | #111827 | Dark text |

### Semantic Colors

| Name | Hex | Usage |
|------|-----|-------|
| Success | #10B981 | Positive states, confirmations |
| Warning | #F59E0B | Caution states |

## Typography

### Font Families

- **Primary:** Inter (or SF Pro, Space Grotesk)
- **Monospace:** JetBrains Mono (for code/technical)

### Type Scale

| Name | Size | Weight | Usage |
|------|------|--------|-------|
| Display | 48-72px | Bold | Hero headlines |
| H1 | 36-48px | Bold | Page titles |
| H2 | 28-36px | Semibold | Section headers |
| H3 | 20-24px | Semibold | Card titles |
| Body | 16px | Regular | Paragraph text |
| Small | 14px | Regular | Secondary text |
| Caption | 12px | Regular | Labels, metadata |

### Guidelines

- High contrast for accessibility
- Generous line height (1.5-1.7 for body)
- Clear hierarchy between heading levels

---

## Iconography

### Style Guidelines

- **Style:** Minimal line icons
- **Stroke:** 1.5-2px consistent weight
- **Corners:** Rounded (2-4px radius)
- **Size:** 16px, 20px, 24px, 32px grid

### Key Icons

| Concept | Icon |
|---------|------|
| Privacy/Security | Lock, Shield, Eye-off |
| Wellness | Heart, Activity, Brain |
| Individual | User, Smartphone |
| Professional | Briefcase, Stethoscope, Dumbbell |

---

## Imagery

### Photography Style

- **Tone:** Warm, natural, authentic
- **Subjects:** Diverse individuals, active lifestyle
- **Setting:** Clean, minimal backgrounds
- **Avoid:** Generic stock photos, overly posed shots

### Illustration Style

- Abstract, geometric patterns
- Soft gradients (navy → violet → cyan)
- Neural network / node patterns for security
- Minimal, line-based diagrams

---

## Animation

### Timing

- **Fast:** 150ms — Micro-interactions
- **Normal:** 300ms — Standard transitions
- **Slow:** 500ms — Page transitions

### Easing

- **Default:** cubic-bezier(0.4, 0, 0.2, 1)
- **Enter:** cubic-bezier(0, 0, 0.2, 1)
- **Exit:** cubic-bezier(0.4, 0, 1, 1)

### Guidelines

- Subtle, purposeful animations
- Avoid excessive motion
- Respect reduced-motion preferences
