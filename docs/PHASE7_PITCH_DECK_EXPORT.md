# Wellio Pitch Deck Graphic Export Plan

## PDF-Ready Production Guide

---

## EXPORT SPECIFICATIONS

### Document Settings
- **Page Size:** 1920 × 1080 px (16:9)
- **Color Mode:** RGB
- **Resolution:** 300 DPI (for print), 72 DPI (for screen)
- **Format:** PDF (primary), PPTX (editable)

### File Naming
```
wellio-pitch-deck-v[version]-[date].pdf
wellio-pitch-deck-v1.0-2024-12.pdf
```

## FONT EMBEDDING

### Required Fonts
- Space Grotesk (Bold, SemiBold)
- Inter (Regular, Medium)

### Embedding Instructions
1. Convert text to outlines (for PDF)
2. Embed fonts (for PPTX)
3. Include font files in delivery package
