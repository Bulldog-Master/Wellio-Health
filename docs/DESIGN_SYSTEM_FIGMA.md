# WELLIO — Platform Design System (Figma Structure)

## Page 1: System Foundations

### Frames
- **Colors** — Primary (#7C3AED), Accent (#22D3EE), Background (#0A0A0C), Surface (#121418)
- **Typography** — Inter family, weights 400/500/600/700
- **Spacing** — 4px base unit, 8/16/24/32/48/64 scale
- **Shadows** — Subtle glow effects with accent gradient
- **Grid system** — 12-column responsive, 1600px max-width

## Page 2: Components

### Variants
- **Buttons** — primary, secondary, destructive, ghost
- **Inputs** — text, numeric, tags, dropdown
- **Pills / Badges** — FWI, trend, flags, tier badges
- **Patient card** — FWI score, trend arrow, adherence flags
- **Clinician overview card** — patient count, session stats
- **Video session row** — scheduled/active status
- **Secure message row** — encryption indicator
- **Side navigation** — collapsed/expanded states
- **Professional tier badges** — Coach Pro/Team, Clinician Individual/Group
- **Smartphone chrome frame** — iOS/Android variants
